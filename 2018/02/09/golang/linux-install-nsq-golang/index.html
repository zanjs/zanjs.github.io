基础概念

  • nsqlookupd: 管理nsqd节点拓扑信息并提供最终一致性的发现服务的守护进程

  • nsqd: 负责接收、排队、转发消息到客户端的守护进程，并且定时向nsqlookupd服务发送心跳

  • nsqadmin：nsqd的web统计界面，可实时查看集群的统计数据和执行一些管理任务
